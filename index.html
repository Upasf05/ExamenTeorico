50 Términos Fundamentales de Desarrollo Web

Nº Término Definición
1HTMLLenguaje de marcado que estructura el contenido de la web mediante etiquetas.
2CSSLenguaje de estilos que define la apariencia visual de elementos HTML.
3JavaScriptLenguaje de programación que permite interactividad en sitios web.
4FrontendParte visible de un sitio web con la que interactúa el usuario.
5BackendParte lógica del desarrollo que corre en el servidor.
6ServidorComputadora que aloja y entrega recursos web a los usuarios.
7HostProveedor de servicios que permite alojar sitios web.
8DominioNombre único que identifica a un sitio web.
9URLDirección que indica la ubicación de un recurso en la web.
10HTTPProtocolo de transferencia de datos entre cliente y servidor.
11HTTPSVersión segura de HTTP con cifrado de datos.
12GETMétodo HTTP para solicitar información al servidor.
13POSTMétodo HTTP para enviar datos al servidor.
14issetFunción de PHP para verificar si una variable está definida.
15echoInstrucción de PHP para mostrar datos en pantalla.
16formElemento HTML que permite enviar datos del usuario.
17inputElemento HTML para entradas de usuario.
18divContenedor genérico en HTML.
19idAtributo HTML que identifica de forma única un elemento.
20classAtributo HTML que permite agrupar elementos para aplicar estilos comunes.
21APIInterfaz para conectar aplicaciones y compartir datos.
22JSONFormato de datos ligero para intercambio entre sistemas.
23AJAXTécnica para actualizar contenidos sin recargar la página.
24PHPLenguaje de programación del lado del servidor.
25SQLLenguaje para gestionar bases de datos relacionales.
26MySQLSistema de gestión de bases de datos relacional.
27BootstrapFramework CSS para diseños responsivos rápidos.
28Media QueryRegla CSS que adapta estilos a diferentes dispositivos.
29ResponsiveDiseño web que se ajusta a distintos tamaños de pantalla.
30SEOPrácticas para mejorar el posicionamiento en buscadores.
31Meta TagEtiqueta HTML con información sobre el documento.
32CacheAlmacenamiento temporal para acelerar cargas.
33CookieArchivo pequeño que guarda información del usuario.
34SessionVariable que mantiene datos mientras el usuario navega.
35FrameworkConjunto de herramientas para desarrollar aplicaciones.
36CMSSistema para crear y gestionar contenidos web.
37WordPressCMS popular para blogs y sitios informativos.
38GitSistema de control de versiones distribuido.
39GitHubPlataforma para colaborar en proyectos con Git.
40RepositoryAlmacén de código y archivos de un proyecto.
41CommitAcción para guardar cambios en Git.
42BranchLínea paralela de desarrollo en Git.
43MergeProceso de unir ramas en Git.
44404Error cuando una página no se encuentra.
45500Error del servidor que impide mostrar la página.
46W3COrganismo que define los estándares web.
47ViewportÁrea visible del navegador donde se muestra el contenido.
48FaviconIcono pequeño que representa una web en el navegador.
49navEtiqueta HTML para agrupar enlaces de navegación.
50footerElemento HTML que contiene el pie de página.
