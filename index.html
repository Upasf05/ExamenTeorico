50 Términos Fundamentales de Desarrollo Web

Nº Término Definición
1 HTML Lenguaje de marcado que define la estructura básica de una página web mediante etiquetas. Es el esqueleto de todo sitio en internet.
2 CSS Lenguaje de hojas de estilo que se utiliza para controlar el diseño, colores, márgenes, tipografía y disposición visual de una web.
3 JavaScript Lenguaje de programación del lado del cliente que permite agregar interactividad, manipular el DOM y conectar con servidores.
4 Frontend Parte visible de un sitio web con la que el usuario interactúa directamente, incluyendo estructura, diseño y comportamiento.
5 Backend Lógica del servidor que gestiona bases de datos, autenticaciones, seguridad y ejecución de procesos del sitio web.
6 Servidor Equipo que almacena archivos web y responde a las solicitudes de los usuarios enviando el contenido al navegador.
7 Host Servicio que permite alojar sitios web en servidores conectados a Internet para que sean accesibles al público.
8 Dominio Nombre único que se utiliza para identificar y acceder a un sitio web, como ejemplo.com o google.com.
9 URL Dirección que identifica un recurso específico en Internet, incluyendo el protocolo, el dominio y la ruta.
10 HTTP Protocolo que permite la transferencia de información entre el navegador del usuario y el servidor web.
